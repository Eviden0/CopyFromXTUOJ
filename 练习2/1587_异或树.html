异或树#

题目描述#

一棵树，节点编号\(1\sim n\)，节点\(1\)是根节点。每个节点有一个初始值，为\(0\)或者\(1\)。

每次你可以执行一个操作，选择任意一棵子树，将其所有节点值进行异或操作，即将\(0\)变成\(1\)，\(1\)变成\(0\)。

请问至少执行多少次操作，可以使得整棵树的节点值是一样的，即都为\(0\)或者都为\(1\)。

输入格式#

第一行是一个整数\(T~(1 \le T \le 100)\)，表示样例的个数。

每个样例的第一行是一个整数\(n~(1 \le n \le 10^4)\)，表示节点数。 以后的\(n-1\)行，每行两个整数\(u,v~(1 \le u \neq v \le n)\)，表示节点\(u,v\)之间存在边。 然后的一行，包含\(n\)个\(0\)或者\(1\)，第\(i\)个数表示节点\(i\)的初始值。

输出格式#

依次每行输出一个样例的结果，为一个整数。

样例输入#

2
3
1 2
1 3
1 0 1
4
1 2
1 3
3 4
0 1 1 0

样例输出#

1
3

样例解释#

  • 第一个样例： 把节点\(2\)翻转，即可得到全是\(1\)的树。

  • 第二个样例： 把节点\(2\)翻转，再把节点\(3\)翻转，再把节点\(4\)翻转，可得到全是\(0\)的树。

提示#

巨大的输入数据，请使用stdio.h头文件和C风格的输入输出。
