火车与汽车#

题目描述#

\(n\)个城市，标号为\(1\sim n\)，城市之间存在铁路或者公路，每条路都可以认为是双向的。

路使用一个四元组\((u,v,w,k)\)表示，表示城市\(u,v\)之间存在一条需要\(w\)时间通过的路，\(k=0\)表示这条路是公路，\(k=1\)表示这条路是铁路。 每次切换公路和铁路，即从铁路转到公路，或者从公路转到铁路，需要额外的\(1\)时间。

请求城市\(1\)到其他所有城市的最少时间。

输入格式#

第一行是一个整数\(T~(1 \le T \le 10)\)，表示样例的个数。

每个样例的第一行是两个个整数\(n~(1 \le n \le 10000),m~(n-1 \le m \le min(50000,n^2)\)，分别表示城市数与路的总数。

以后的\(m\)行，每行四个整数\(u,v,w,k\)。其中\(1 \le u,v \le n,~1 \le w \le 10000,~0 \le k \le 1\)。

输入保证所有城市都是连通的。

输出格式#

依次每行输出一个样例的结果，为\(n-1\)个整数。

第\(i\)个数表示城市\(1\)到城市\(i+1\)的最短时间。每个整数之间使用一个空格隔开。

样例输入#

2
3 2
1 2 1 0
2 3 2 1
3 3
1 2 1 0
2 3 2 1
2 3 4 0

样例输出#

1 4
1 4

提示#

巨大的输入数据，请使用stdio.h头文件和C风格的输入输出。
