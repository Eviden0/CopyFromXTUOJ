二叉树合并#

题目描述#

二叉树上的每个节点有一个值，请将两个二叉树进行合并。如果只有一个二叉树上有这个节点，那么合并后的节点值为原来节点的值，否则为两个节点值的和。

如下图所示。

图中红色数字为节点序号，节点中蓝色数字为节点权值。

输入格式#

第一行是一个整数\(T(1 \le T \le 100)\)，表示样例的个数。

每个样例包含两个树，每个树的第一行是一个整数\(n (1 \le n \le 100)\)，表示树节点的数量。

以后的\(n\)行，每行三个整数，表示每个节点权值（不超过10000的正整数）和其左右孩子节点的序号，默认序号\(1\)为根节点，如果序号为\(0\)，则表示孩子节点为空。

输出格式#

依次输出每个样例的结果，为合并后二叉树的节点权值的前序和中序遍历结果。第一行是前序遍历，第二行是中序遍历。 权值之间用一个空格隔开，行尾不要多余的空格。

样例输入#

1
4
1 2 3
2 4 0
5 0 0
4 0 0
4
2 2 0
2 3 4
3 0 0
1 0 0

样例输出#

3 4 7 1 5
7 4 1 3 5
