工作#

题目描述#

有\(n\)件工作，分给小明和小华完成，他们对于第\(i\)件工作完成所需的时间分别为\(a_i\)和\(b_i\)。一件工作只能由其中某一人单独完成，个人完成的时间是分配的工作所需时间的总和。求最少需要多少时间能完成所有的工作？

输入格式#

第一行是一个整数\(T~(1 \le T \le 100)\)，表示样例的个数。

每个样例的第一行是一个整数\(n~(1 \le n \le 1000)\)，第二、三行都为\(n\)个正整数，第二行为\(a_i\)，第三行为\(b_i\)，\(1 \le a_i,b_i \le 1000\)。\(\sum_{i=1}^n{a_i} \le 10^5\) 或者\(\sum_{i=1}^n{b_i} \le 10^5\)。

输出格式#

样例输入#

2
3
1 2 1
2 2 2
3
1 2 3
3 2 1

样例输出#

2
3

样例解释#

第一个样例中，工作\(1,3\)给小明，工作\(2\)给小华，小明需要\(2\)单位时间，小华也需要\(2\)单位时间完成，所以最终完成时间是\(2\)。

第二个样例中，工作\(1,2\)给小明，工作\(3\)给小华，小明需要\(3\)单位时间，小华也需要\(1\)单位时间完成，所以最终完成时间是\(3\)。
