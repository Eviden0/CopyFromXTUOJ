前缀和

题目描述

你有两个数列\(A=\{a_1,a_2,\cdots,a_n\}\)和\(B=\{b_1,b_2,\cdots,b_m\}\)。你可以将\(B\)中的元素，移动到数列\(A\)的任意位置，得到一个新的数列\(C=\{c_1,c_2,\cdots,c_{n+m}\}\)。

我们定义前缀和函数\(f(k)=\sum_{i=1}^{k}c_i\)，求最大\(k\)，使得\(f(k)<0\)。

比如\(A=\{-4,3,1\},~B=\{4,2,1\}\),我们可以将\(B\)中的元素移动\(A\)数列中，得到新的数列\(C=\{1,2,-4,4,3,1\}\)，此时\(k=3\)时，\(f(3) = -1\)，可以发现无论哪种移动方案，使得\(f(k)<0\)时，最大的\(k=3\)。所以结果为\(3\)。

输入格式

第一行是一个整数\(T (1 \le T \le 300)\)，表示样例的个数。

每个样例的第一行是两个整数\(n,m,~(1 \le n,m \le 1000)\)；第二行是\(n\)个元素\(a_i\)；第三行是\(m\)个元素\(b_j\)；所有元素的绝对值都不超过\(10^5\)。

输出格式

依次每行输出一个样例的结果，为一个整数。如果不存在这样的\(k\)，使得\(f(k)<0\)，输出\(0\)。

样例输入

3
3 2
1 2 3
1 2
3 2
1 2 3
-1 -2
3 3
-4 3 1
4 2 1

样例输出

0
3
3

样例解释

第一个样例，无论我们怎么处理，前缀和都不小于0；

第二个样例，我们可以合并得到数列\(\{-1,-2,1,2,3\}\)，此时\(f(3) = -2 < 0\)，所以结果为\(3\)。
