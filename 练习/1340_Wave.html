题目描述

一个$n$列的网格，从$(0,0)$网格点出发，波形存在平波(从$(x,y)到(x+1,y)$)，上升波(从$(x,y)$到$(x+1,y+1)$)，下降波(从$(x,y)到(x+1,y-1)$)三种波形，请问从$(0,0)$出发，最终到达$(n,0)$的不同波形有多少种？如图，$3$列网格有7种不同的波形。

输入

第一行是样例数$T(1 \le T \le 42)$。 以后每行一个整数$n(1 \le n \le 42)$。

输出

每行输出一个样例的结果。

样例输入

3
1
2
3

样例输出

1
3
7
